# Pruebas Automatizadas - Plataforma Virtual ITLA

## Requisitos
- Python 3
- Google Chrome
- ChromeDriver compatible
- Instalar dependencias:
```bash
pip install -r requirements.txt - si no funciona simplemente instale las dependencias manualmente
```

## Ejecutar pruebas
```bash
pytest --html=report.html --self-contained-html
o
pytest
```

Las capturas se guardarán en `screenshots/` y el reporte en `report.html`.

## Credenciales
Están configuradas en el archivo `.env` para seguridad.
